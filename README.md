# 后台管理系统

> 一个基于node.js与vue的后台管理系统，实现基本的CRUD以及jwt鉴权。

## 项目截图

- 主页

![主页](https://github.com/mogu396/myImgStorage/blob/main/vue-management-project.png?raw=true)


## 用到的框架以及插件
#### 前端
- vue
- vue Router
- vuex
- Axios
- ElementUI
  
#### 后端
- express(4.x)
- mongodb(mongoose)
- jsonwebtoken


## 项目运行设置

- 克隆到本地

```
git clone https://github.com/mogu396/vue-management-project
```

- 分别进入前后端项目
  
```
yarn
```

- 启动项目

```
yarn serve
```
